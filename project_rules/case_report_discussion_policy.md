# Case Report Discussion Policy

この指示簿は、医学症例報告の考察を作成・修正するときにCodexが必ず参照するルールです。

## 基本方針

考察は文献紹介ではなく、本症例で生じた臨床判断の問いに答える文章として構成する。

悪い考察は、文献A、文献B、文献Cを順に紹介してから本症例を添える形になりやすい。良い考察は、まず本症例で何が問題だったかを示し、その問題を説明するために文献を使う。

## 最初に決める問い

考察を書く前に、以下を1文で定義する。

```text
本症例で最も難しかった臨床判断は何か。
```

例:

```text
腎温存が標準的に重視される小児腎損傷において、どの時点で温存治療の継続が患者にとって不利益へ転じたのか。
```

この問いが考察全体の背骨になる。

## 推奨構成

### 1. 本症例の特徴と新規性

冒頭では、症例の要点ではなく「なぜ報告する価値があるか」を明確にする。

書くべき内容:

- ガイドラインや標準治療に沿った初期対応を行ったこと
- それでも制御困難だった点
- 本症例が示す臨床的問い

避けること:

- 症例経過の単なる再掲
- 「稀である」とだけ書くこと

### 2. 初期治療選択の妥当性

初期判断が妥当だったかを文献で確認する。

重要なのは、初期治療を正当化したうえで、次の問題へ橋渡しすることである。

例:

```text
したがって、本症例の問題はNOMを選択したことではなく、NOM継続中に制御不能な溢尿へ移行した後の判断にあった。
```

### 3. 通常経過から逸脱した点

既報と本症例の共通点だけでなく、相違点を書く。

書くべき内容:

- 既報ではどこまでが通常の範囲か
- 本症例では何が通常管理から外れていたか
- なぜそれが治療上の問題になったか

例:

```text
尿漏そのものは高グレード腎損傷で起こり得るが、本症例では標準的な尿路減圧とドレナージ後も多量の尿漏出、感染、尿管狭窄、長期チューブ依存が持続した点が異なっていた。
```

### 4. 追加治療を病態生理で説明する

処置を羅列しない。各治療を、何を解決するための戦略だったかで整理する。

例:

- 漏出源となる腎実質の尿産生を減らす
- 産生された尿を別経路へ逃がす
- 残存瘻孔を閉鎖する

このように治療意図を明確にすると、医学論文としての論理が立つ。

### 5. 治療が奏効しなかった理由を慎重に考察する

症例情報に基づいて、推測しすぎずに機序を述べる。

書いてよい例:

```text
損傷腎盂が広範であったこと、尿管狭窄により下流抵抗が残存したこと、感染・炎症により瘻孔閉鎖が妨げられたことが、溢尿遷延に関与した可能性がある。
```

避けること:

- 症例情報にない解剖学的詳細を断定する
- 文献で確認していない機序を一般論として書く
- 「明らかである」と過度に断定する

### 6. 臨床判断の主題に戻る

考察後半では、最初に定義した臨床判断の問いへ戻る。

臓器温存の利益だけでなく、温存治療継続の不利益を評価する。

評価軸の例:

- 排液量の推移
- 感染制御の可否
- 尿路狭窄の有無
- チューブ依存期間
- 追加処置の見込み
- 患者・家族の受容可能性
- 小児では学業、家族・友人関係、隔離、治療終期不明による心理社会的負担

### 7. 小児心理社会的負担の扱い

小児精神面の文献は、独立した一般論として置かない。治療継続のリスクベネフィット判断に統合する。

良い使い方:

```text
本症例では、文献が示す長期入院後の心理社会的リスクを背景に、治療終期の見通しが立たないチューブ依存生活が本人に及ぼす不利益を、腎温存継続のリスクとして評価した。
```

避ける使い方:

```text
小児入院では精神的問題が起こることが知られている。
```

### 8. 臨床的示唆と限界

最後は「本症例から何を学ぶか」に戻る。

限界は短く書く。限界で考察全体を終わらせる場合も、直前または直後に臨床的示唆を明確にする。

## 段落作成ルール

- 1段落1テーマにする。
- 各段落の冒頭で、その段落の役割を示す。
- 各段落の末尾に、次段落への橋渡し文を置く。
- 「文献では」で始まる段落を連続させない。
- 文献の数値紹介は最小限にし、本症例との差分を中心に書く。
- 「本症例は〜を示唆する」を多用しない。重要な結論に絞って使う。
- 処置名の羅列ではなく、治療意図と病態生理で整理する。

## 引用の使い方

引用は、本文の主張を支えるために使う。

引用すべき箇所:

- 標準治療やガイドラインに関する記載
- 頻度、予後、合併症に関する記載
- 既報症例や類似手技との比較
- 小児長期入院・外傷後の心理社会的影響

引用しない方がよい箇所:

- 入力症例情報だけで足りる事実
- 文献と直接対応しない推測
- 一般論として薄い文章

## 最終確認

考察を書いた後、以下を確認する。

- 冒頭段落だけで本症例の報告価値が分かるか。
- 各段落が臨床判断の問いに戻っているか。
- 文献紹介ではなく、本症例との差分を説明しているか。
- 処置の羅列になっていないか。
- 小児心理社会的負担が治療判断に統合されているか。
- 最終段落で本症例から得られる実践的示唆が明確か。
