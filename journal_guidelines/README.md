# Journal Guidelines

このフォルダーには、投稿先雑誌の投稿規定を保存します。

## 推奨ファイル

- `README.md`: 投稿規定URL、確認日、主要規定の要約
- `author_guidelines.md`: 投稿規定本文から作業に必要な範囲を転記した確認メモ
- `author_guidelines.pdf`: 投稿規定PDFを保存する場合の原本

## 記載テンプレート

```text
## 雑誌名

## 投稿規定URL

## 確認日

## 論文種別

## 主要規定

- 要旨:
- 本文文字数:
- 図表数:
- キーワード:
- 文献形式:
- 倫理・患者同意:
- COI:

## 注意

- URLのみでなく、確認した主要規定をこのファイルに要約する。
- 投稿前に最新規定を再確認する。
- Codexはここに保存された投稿規定またはユーザーが提示した本文を優先する。
```

## 運用ルール

- 投稿規定がURLのみの場合でも、確認日と主要規定メモを必ず残す。
- 規定本文を確認できない場合、Codexは推測で補完せず `要確認` として扱う。
- 古いJ-STAGE掲載規定を使う場合は、現行投稿システムとの差分確認を残課題にする。
